# Bootstrap Website (AWS Amplify Hosting)

This project is a simple personal website built using Bootstrap and hosted using AWS Amplify.

## About
This website was created as part of an assignment to practice building and hosting a static website using GitHub and AWS.

The site includes:
- A Bootstrap layout
- Personal information
- An image
- Basic responsive design

## Technologies Used
- HTML
- Bootstrap
- GitHub (repository)
- AWS Amplify (hosting)

## How to Run Locally
1. Download or clone the repository
2. Open the folder in VS Code
3. Open `index.html` in a browser

## Deployment
This site is hosted using AWS Amplify.

Any changes pushed to the GitHub repository will automatically update the live site.
